# Multi Agent Framework

## Introduction

Multi Agent Framework can do talk and task and self-play with multiple agents.


## Pre-requirements

```bash
pip install -r requirements.txt
cp .env_example .env
```

write your own KEY and URL(optional)

```
OPENAI_API_KEY=xxx
OPENAI_BASE_URL=xxx
```
