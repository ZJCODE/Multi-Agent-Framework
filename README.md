# Multi Agent Framework

## Introduction

The Multi-Agent Framework can communicate, execute tasks, and manage low-level control.

## Pre-requirements

```bash
pip install -r requirements.txt
cp .env_example .env
```

write your own KEY and URL(optional)

```
OPENAI_API_KEY=xxx
OPENAI_BASE_URL=xxx
```
